# Banks

Retrieve a list of all supported banks and financial institutions in Nigeria.

## List All Banks

Get list of banks and financial institutions.

<Callout>
**GET** `https://api.lenco.co/access/v1/banks`
</Callout>

### Response Schema

```json
{
  "status": true,
  "message": "string",
  "data": [
    {
      "code": "string",
      "name": "string"
    }
  ]
}
```

### Example Request

```bash
curl --request GET \
  --url https://api.lenco.co/access/v1/banks \
  --header 'Authorization: Bearer YOUR_API_TOKEN' \
  --header 'accept: application/json'
```
