# /resolve/bank-account

POST https://api.lenco.co/access/v2/resolve/bank-account

Use this endpoint to verify and resolve bank account details. Required body parameters include `accountNumber` and `bankId`.

### Request body

- accountNumber (string) - required
- bankId (string) - required
- country (string) - optional, i.e. `ng`, `zm`

### Response schema

```json
{
  "status": boolean,
  "message": string,
  "data": {
    "type": "bank-account",
    "accountName": string,
    "accountNumber": string,
    "bank": { "id": string, "name": string, "country": string }
  }
}
```
