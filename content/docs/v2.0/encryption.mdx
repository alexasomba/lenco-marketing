# Encryption

Lenco uses RSA and AES for secure payload encryption. Obtain an RSA public key from the `GET /encryption-key` endpoint and use JWE to wrap your payload.

Follow canonical examples on the original ReadMe for concrete implementations using Go, Node and other languages.
