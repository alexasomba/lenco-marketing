# /banks

GET https://api.lenco.co/access/v2/banks

Retrieve a list of banks and financial institutions.

### Query params

- country — optional (e.g., `ng`, `zm`)

### Response schema

```json
{
  "status": boolean,
  "message": string,
  "data": [ { "id": string, "name": string, "country": string } ]
}
```
