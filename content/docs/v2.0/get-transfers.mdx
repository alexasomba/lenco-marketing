# /transfers

GET https://api.lenco.co/access/v2/transfers

Return transfers and use filters (accountId, status, type). See endpoint-specific subpages for POST operations (bank-account, mobile-money, lenco-money, lenco-merchant, account).

Response schema examples are provided on the ReadMe page.
