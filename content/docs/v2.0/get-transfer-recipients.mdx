# /transfer-recipients

GET https://api.lenco.co/access/v2/transfer-recipients

Retrieve information about transfer recipients. Supports pagination and filters.

### Response schema

```json
{
  "status": boolean,
  "message": string,
  "data": [ { "id": string, "currency": string, "type": string, "country": string, "details": {} } ],
  "meta": { "total": number, "perPage": number, "currentPage": number, "pageCount": number }
}
```
